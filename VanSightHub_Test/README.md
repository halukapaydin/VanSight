# VanSightHub Test Projesi

Bu proje VanSightHub'a ESP-NOW komutları göndermek için kullanılır.

## Kurulum

1. Yeni bir ESP32 kartı hazırlayın
2. Bu dizindeki kodu yükleyin
3. Serial monitörden test komutlarını gönderin

## Kullanım

Serial monitörde şu komutları yazabilirsiniz:
- `1` - 1. röleyi aç
- `2` - 1. röleyi kapat
- `3` - Tüm röleleri aç
- `4` - Tüm röleleri kapat
- `5` - Tüm durum bilgisi al
- `6` - Sensör 1 oku
- `7` - Tüm sensörleri oku
